# JavaBackend 面试题库网站

一个现代化的Java后端面试题库网站，采用主流科技网站设计风格，支持响应式布局、深色模式、搜索功能。

## 特点

- 🎨 **现代化设计** - 采用渐变背景、圆角卡片、阴影分层，符合当前主流设计风格
- 🌓 **深色模式** - 支持浅色/深色主题切换，自动保存偏好
- 🔍 **全文搜索** - 支持关键词搜索所有面试题
- 📱 **响应式** - 完美适配移动端和桌面端
- 🎯 **分类清晰** - 按知识模块分类，支持知识体系图谱展示
- ⚡ **纯静态** - 无需后端，直接打开index.html即可运行

## 目录结构

```
java-interview-website/
├── index.html      # 主页面
├── style.css       # 样式文件
├── script.js       # JavaScript逻辑
└── README.md       # 说明文档
```

## 使用方法

直接用浏览器打开 `index.html` 即可访问网站。

如果需要本地服务运行：

```bash
# 使用Python启动简易服务器
python3 -m http.server 8000

# 然后访问 http://localhost:8000/java-interview-website/
```

## 功能模块

### 题库分类
- Java基础
- JVM原理
- 并发编程
- Spring全家桶
- 数据库
- 缓存技术
- 分布式系统
- 系统设计

### 特色功能
- 点击分类卡片查看该分类下的面试题
- 点击搜索按钮可以关键词搜索所有题目
- 支持折叠展开知识体系分类
- 滚动自动高亮导航
- 点击回到顶部按钮

## 扩展说明

如果需要接入更多题目，可以在 `script.js` 中的 `questions` 对象继续添加题目数据。如果需要后端API，可以修改JavaScript代码从后端接口获取题目数据。
