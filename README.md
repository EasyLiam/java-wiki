# Java 知识库

Java后端开发面试知识整理，系统化梳理，每日自动更新，助力你更好地准备面试。

## 特性

- 📚 **系统化整理** - 按主题和难度分级
- 📅 **每日更新** - 每天自动搜索更新题库
- 🔍 **全文搜索** - 支持搜索任何问题
- 📱 **响应式设计** - 完美适配移动端
- 🎨 **现代美观** - 使用 VitePress 构建，体验流畅

## 开发启动

```bash
# 安装依赖
npm install

# 本地开发
npm run docs:dev

# 构建生产版本
npm run docs:build
```

## 自动更新

该知识库每天自动搜索新的面试题并更新到对应主题，保持题库新鲜。
